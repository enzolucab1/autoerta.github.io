Sistema informatico

La idea de la huerta es que sea automatizada, que el consumidor sólo tenga que poner la tierra y las semillas. Para esto necesitamos que el sistema haga todo el resto del trabajo por su propia cuenta. Para que esto sea posible es necesario que, primero, interprete el estado de humedad, por lo que se utiliza un sensor de humedad de suelo, el cual mide el porcentaje de agua presente en la tierra, que controla el sensor en la superficie. Este sensor interpreta e identifica los datos obtenidos para enviar una señal y “avisar” si se necesita o no agua. El agua es proporcionada por una bomba de agua pequeña, que a su vez necesita de un servomotor, este actuador requiere de energía para funcionar y controlar el flujo de agua a un determinado nivel, por lo que utilizamos una Pila de 9V para alimentarlo. Para que este proceso tenga señales físicas, incluímos un LED que nos indique el nivel de humedad en la huerta. Cuando el nivel de humedad sea menor a 300, quiere decir que la huerta está seca y necesita riego, y el LED es de color azul. Cuando el nivel está entre 301 y 700, el LED será de color verde y la huerta tendrá la humedad adecuada para su funcionamiento. Pero si este nivel es mayor de 700, significa que se encuentra en exceso de riego, y el LED es de color rojo. El LED necesita de un relé SDPT, que funciona como un interruptor.
