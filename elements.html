Programacion

Para la parte de programacion diseñamos una pagina web la cual tenía distintas secciones explicativas en base al paso a paso desarrolado en cada materia incluida en este proyecto.

Para diseñar la pagina web utilizamos el programa Visual Studio Code basandonos en un codigo de programacion existente el cual fue copiado y editado por nosotros para llegar al diseño web que deseabamos.
Para agregar las secciones utilizamos la funcion "elements list" (< li>).
Luego agregamos las imagenes utilizando la funcion < img src=(link de la imagen) alt="">
A la hora de agregar texto se uso la funcion < p> (texto) < p>

Esta pagina web se realizo con el fin de explicar el paso a paso llevado a cabo en cada materia, ademas se trabajo para que sea lo mas rapida/sencilla posible.
